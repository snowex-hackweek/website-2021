Getting ready¶

We will open the workshop with getting connected to the hackweek JupyterHub. https://snowex.hackweek.io.

Please take a minute before the workshop to go over the GitHub instructions , create an account, and sent your user name to snowex@uw.edu so we can add you to the SnowEx hackweek GitHub organization.

We will be using the JupyterHub (https://snowex.hackweek.io) throughout this workshop.
